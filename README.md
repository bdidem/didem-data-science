## Projects

###### 01 — E-commerce KPI \& Retention Analysis



Goal: Understand user behavior, conversion funnel, and retention dynamics.

Highlights:

&#x09;SQL-based KPI analysis

&#x09;Funnel analysis (view → add-to-cart → purchase)

&#x09;Cohort retention analysis with heatmap

&#x09;Business-driven insights and recommendations

Key Insight:

&#x09;The largest drop-off occurs between product view and add-to-cart, while users who add items to cart convert at a very high rate (\~95%).



###### 02 — Repeat Purchase Prediction (Churn Modeling)



Goal: Predict whether a user will make a repeat purchase after their first order.

Highlights:

&#x09;Feature engineering from transactional and behavioral data

&#x09;Careful handling of data leakage

&#x09;Logistic Regression baseline + Random Forest model

&#x09;Model evaluation (accuracy, ROC-AUC) and feature importance analysis

Key Insight:

&#x09;Behavioral engagement signals (especially add-to-cart activity) are stronger predictors of repeat purchase than static user attributes.
